Lista de questões - Lógica da Programação

Atividade sobre comandos e funcionalidades do GIT.

O que é algoritmo? Explique com suas palavras.

É um guia completo para realizar alguma ação, um passo-a-passo de alguma funcionalidade, atividade ou processo.

Explique o que significa: Entrada, processamento e saída.

Entrada se refere à entrada de dados no dispositivo, processamento é a leitura e reconhecimento dos tais, e saída remete à execução e conteúdo final, este já "refinado" e processado dos dados originais.

O que é uma estrutura sequencial de programação?

Instruções executadas uma após a outra, na ordem em que aparecem, sem desvios ou repetições.

O que é uma variável? Dê um exemplo.

Variável é um espaço usado para guardar um valor que pode mudar. Exemplo: idade = 18 (a variável “idade” guarda o valor 18).

Cite três tipos de dados utilizados na programação e explique cada um.

Inteiro (int): guarda números sem parte decimal. Ex: 10
Real/Float: guarda números com vírgula (decimais). Ex: 3.5
Texto (String): guarda palavras ou frases. Ex: "Olá"

Para que serve uma estrutura condicional?

Uma estrutura condicional serve para tomar decisões no programa. Ela permite que o código execute ações diferentes dependendo de uma condição.

Dê exemplo de uma situação no dia-a-dia que utilize uma decisão (SE).

Exemplo do dia a dia: Se estiver chovendo, levo guarda-chuva; se não, saio sem.

O que é uma estrutura de repetição?

Uma estrutura de repetição é usada para executar uma ação várias vezes. Ela repete um bloco de código enquanto uma condição for verdadeira ou por um número definido de vezes.

Dê um exemplo do cotidiano onde ocorre repetição.

Usar Ctrrl + C e Ctrl + V.

O que são operadores? Cite dois exemplos.

Operadores são símbolos usados para realizar operações com valores. Exemplos: `+` (soma) e `>` (maior que).

Para que serve os operadores relacionais?

Os operadores relacionais servem para comparar valores e retornar verdadeiro ou falso.

O que significa o valor booleano?

O valor booleano é um tipo de dado que só pode ter dois valores possíveis: Verdadeiro (True) ou Falso (False). Ele é usado principalmente em decisões e comparações, como em estruturas condicionais e de repetição.

O que é uma expressão lógica?

Uma expressão lógica é uma condição que pode ser verdadeira ou falsa. Ela combina valores e operadores lógicos para ajudar o programa a decidir o que fazer.

Para que serve um fluxograma?

Um fluxograma serve para representar visualmente o passo a passo de um processo ou programa. Ele ajuda a entender, planejar e organizar as etapas antes de escrever o código, usando símbolos para decisões, ações e início/fim.

O que é um pseudocódigo?

Um pseudocódigo é uma forma de escrever a lógica de um programa usando linguagem simples, parecida com português, sem se preocupar com a sintaxe de uma linguagem de programação. Ele serve para planejar e organizar o código antes de programar de verdade.

Qual a diferença entre um algoritmo e um programa?

Um pseudocódigo é uma forma de escrever a lógica de um programa usando linguagem simples, parecida com português, sem se preocupar com a sintaxe de uma linguagem de programação. Ele serve para planejar e organizar o código antes de programar de verdade.

O que é um erro lógico?

Um erro lógico é quando o programa roda sem travar, mas não produz o resultado correto porque a lógica usada está errada.

Por que a lógica da programação é importante?

A lógica de programação é importante porque ajuda a pensar de forma organizada para que o programa funcione corretamente. Ela permite resolver problemas passo a passo, evitar erros e criar códigos eficientes e fáceis de entender. Sem lógica, o programa pode não funcionar ou produzir resultados errados.

Por que um algoritmo precisa ser escreito em ordem?

O algoritmo precisa estar em ordem porque cada passo depende do anterior. Se a sequência estiver fora de ordem, o programa pode dar erro ou produzir resultados errados.
